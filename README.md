## 準備步驟
- 下載 Discord_bot.py
- 安裝 discord.py 輸入“pip install discord.py”
- 下載 Visual Studio Code (VScode) 或其他 IDE
## 如何使用
- “YOUR_BOT_TOKEN”更改為您自己的令牌
- 按下“開始”按鈕！
